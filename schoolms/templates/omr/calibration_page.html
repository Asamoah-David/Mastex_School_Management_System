How to calibrate
  1. Upload a blank sheet image (or a clear scan) — same paper size as real exams.
  2. Drag each dashed rectangle so it covers only the inner fill area of each bubble (not brackets or letters).
  3. Drag the red line to match the vertical spacing to the next question row (baseline).
  4. Optional: Move whole column keeps A–E spacing while shifting a column sideways.
  5. Save — coordinates are scaled to the template ({{ tmpl.image_width }}×{{ tmpl.image_height }} px).

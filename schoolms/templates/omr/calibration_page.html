Calibration Guide

This page helps you calibrate the OMR detection coordinates for your specific printer/scanner setup.

  1. Print the answer sheet template using your standard printer
  2. Fill in a sample answer sheet with known answers
  3. Upload a clear, well-lit photo or scan of the sheet
  4. Mark the reference points on the image to calibrate coordinates
Note: Calibration should only be needed if you're using a non-standard sheet layout or if detection is consistently misaligned.
